charAt()方法

2020-08-20 1 min read
  • charAt() 方法用于返回指定索引处的字符。索引范围为从 0 到 length()-1。
public char charAt(int index)       // index为字符的索引
  • 实例：
public class Main{
    public static void main(String[] args) {
        String s = "www.jack.com";
        char c = s.charAt(5);
        System.out.print(c);
    }
}
