使用 navicat 连接 Linux 的 mysql 出现的 1251 错误代码

  1. 首先进入容器
docker exec -it mysql01 bash  (mysql01为容器名)
  1. 登陆mysql
mysql -u root -p
Enter password:
  1. 修改加密规则
use mysql;

# 最后引号是密码，%代表任意ip，也可修改为某个ip
ALTER USER 'root'@'%' IDENTIFIED WITH mysql_native_password BY '123456';

# 刷新权限表
flush privileges;
