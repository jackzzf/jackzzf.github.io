访问Tomcat容器报错解决办法

2020-08-10 1 min read
  • 启动Tomcat容器
docker run -d -p 8888:8080 tomcat
  • 查看已启用的容器
docker ps
  • 查看防火墙状态
service firewalld status
  • 若开启，则关闭
service firewalld stop
  • 此时再访问还是报错：则说明webapps文件夹里面没有内容
docker exec -it 容器id /bin/bash

ls -l

cd webapps

ls -l

# 文件夹内没有内容，显示total 0

cd ..

rm -rf webapps

mv webapps.dist webapps

# 此时访问即可
